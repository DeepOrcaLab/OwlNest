## Modification Notice

This repository contains substantial modifications to the original Read Frog project.

Major changes include:

- OwlNest branding and owl icon
- AI Reading Agent side panel
- Chat memory and page-aware conversations
- Local Knowledge Nest
- Save to Knowledge workflow
- Multi-provider AI support (DeepSeek, OpenAI, MiMo, OpenRouter)
- Streaming AI responses
- Voice interaction support
- Knowledge import/export
- Removed account/cloud dependency
